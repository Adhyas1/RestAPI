GET /users/(int: user_id)/Books¶

User ( ‘User_id’ API to get a list of books).

Request example:

GET /users/123/web HTTP /1.1
Host: facebook.com
Accept: Application / json

Response example:

HTTP/1.1 200 OK
Vary: Accept
Content - Type: Application / json

[
  {
    "Book_id": 123,
    "author_id": 35,
    "Title": "Sphinx Book"
   }
]
Query Parameters:
 
  • sort – one of hit, created-at
  • offset – offset number. default is 0
  • limit – limit number. default is 30
Request Headers:
 
  • Accept – the response content type depends on Accept header
  • Authorization – optional OAuth token to authenticate
Response Headers:
 
  • Content-Type – this depends on Accept header of request
Status Codes:
  • 200 OK – no error
  • 404 Not Found – there’s no user
